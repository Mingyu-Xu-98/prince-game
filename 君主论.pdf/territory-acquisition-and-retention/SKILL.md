# 领土征服与保有策略

## 适用场景
- 计划征服或已征服一个新国家
- 需要整合新获取的领土
- 需要根据目标国家类型制定保有策略

## 策略选择流程

### 第一步：分析领土同质性

#### 同质混合君主国
**适用条件：**
- 被征服国家与征服国属于同一地区
- 使用同一语言
- 被征服者不习惯于自由生活

**执行步骤：**
1. 灭绝过去统治它们的君主的血脉世系
2. 在其他事情上维持它们旧的状态
3. 不要改变它们的法律或赋税

**预期结果：** 领地将迅速与原有国家融合，且不易发生叛乱

#### 异质混合君主国
**适用条件：** 被征服国家在语言、风俗习惯和秩序上与征服国不同

**执行策略：**

1. **君主亲自驻跸**
   - 动乱稍一露头就能察觉并迅速补救
   - 防止官吏掠夺
   - 臣民可随时求助而感到满意
   - 良民更爱戴，坏人更畏惧
   - 外部进攻者更加迟疑

2. **派遣殖民（推荐）vs 驻军（避免）**
   - 殖民：花费不高，更加忠诚可靠，侵害性小
   - 驻军：靡费甚巨，军队危害整个国家
   - 侵害原则：对人们要么加以安抚，要么加以剪除

3. **地缘政治制衡**
   - 成为邻近弱小势力的首领和保护者
   - 削弱该地区的强大势力
   - 防止同自己一样强大的外国势力介入

4. **预防性干预**
   - 在患病之初就察识并恢复
   - 绝不为了避免一场战争而听任动乱继续
   - 宁愿在自己有能力时在远方作战

### 第二步：分析目标国家统治结构

#### 类型A：由一位君主和臣仆统治（如土耳其）
- **征服难度：** 困难（无法通过收买内部人员进入）
- **保有难度：** 容易（只需击败君主并灭绝其血脉）

#### 类型B：由一位君主和诸侯统治（如法兰西）
- **征服难度：** 相对容易（可收买心怀不满的诸侯）
- **保有难度：** 非常困难（残留领主仍会成为变革首领）

### 第三步：分析政治文化传统

#### 习惯于自由的城市/国家
**风险：** 人民对自由和古老秩序的记忆无法被岁月或恩惠消除

**策略选择：**
1. **毁灭它们**（最安全稳固）
2. **亲自前往生活在那里**
3. **允许它们生活在自己的法律下，但建立寡头政权**（有风险）

#### 习惯于生活在一位君主统治之下
**优势：** 一方面习惯服从，另一方面没有了旧君主
**策略：** 君主可以轻而易举地获得并保有它们

## 关键变量

| 变量 | 类型 | 描述 |
|------|------|------|
| 同质性 | Boolean | 语言、风俗、秩序是否相同 |
| 统治结构类型 | Enum | 君主-臣仆型 或 君主-诸侯型 |
| 政治文化传统 | Enum | 自由共和国 或 世袭君主国 |
| 君主驻在 | Boolean | 君主是否亲自居住在当地 |
| 控制手段 | Enum | 殖民或驻军 |

## 禁忌法则
- 谁是促使他人强大的原因，谁就是自取灭亡
- 不要通过费尽心机或武力使他人变得强大
