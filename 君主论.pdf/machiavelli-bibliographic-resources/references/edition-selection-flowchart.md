# 版本选择流程图

## 翻译工作

```
开始
  ↓
需要翻译《君主论》？
  ↓ 是
选择意大利文原版作为底本
  ↓
学术研究 → 因格莱塞版（1994）或国家版（2006）
  ↓
参考英译本辅助理解
  ↓
曼斯菲尔德版（1985）或剑桥大学版（1988）
  ↓
参考中译本对比
  ↓
潘汉典译本（1985）作为主要参考
```

## 学术研究

```
开始
  ↓
需要研究马基雅维利思想？
  ↓ 是
阅读研究著作
  ↓
思想史 → 迈内克、施特劳斯、波考克、斯金纳
  ↓
传记 → 维罗利、盐野七生、赖因哈特
  ↓
文本疏证 → 阿尔瓦热兹、曼斯菲尔德
  ↓
对比不同译本
  ↓
意大利文原版 + 多个英译本 + 多个中译本
```
