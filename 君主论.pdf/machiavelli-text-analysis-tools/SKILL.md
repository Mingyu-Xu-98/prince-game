# 马基雅维利文本分析工具

## 使用场景
当需要以下情况时应用此技能：
- 查找书中特定人名、地名、民族名的出现位置
- 定位文本中的具体段落
- 分析书信121“佩鲁贾的奇思妙想”中的哲学概念
- 理解马基雅维利对外在力量的整体看法

## 专名索引使用规则

### 索引结构

**排序方式**：索引条目按照汉语拼音排序

**定位格式**：罗马数字为章节号，阿拉伯数字为段落号

### 格式解读

**基本格式**：`条目名：章节号. 段落号`

**示例**：
- `阿尔巴：VI. 3` → 该条目出现在第6章第3段
- `切萨雷·博尔贾：VII. 2, VII. 5, VII. 6` → 该条目分别出现在第7章第2段、第5段、第6段

### 内容覆盖

索引包含以下类型的条目：
- **人名**：如切萨雷·博尔贾、教皇尤利乌斯二世、马基雅维利等
- **地名**：如佛罗伦萨、米兰、罗马等
- **民族名**：如法国人、意大利人、土耳其人等

### 使用流程

1. **确定查找对象**：确定需要查找的人名、地名或民族名
2. **按拼音查找**：在索引中按汉语拼音顺序查找条目
3. **解读位置信息**：根据罗马数字确定章节，阿拉伯数字确定段落
4. **定位文本**：根据位置信息在正文中找到对应内容
5. **多重引用**：注意条目可能在多处出现

**信件编号**：书信121

### 分析方法

**触发条件**：分析书信121中的概念

**执行步骤**：
1. **识别概念**：在文本中识别上述六个核心概念
2. **关联分析**：将不同概念关联起来，理解马基雅维利对外在力量的整体看法
3. **区分层次**：区分偶然性（fortuna, sorte）与必然性（fati）
4. **综合理解**：结合超自然力量（cielo, stele）和时代因素（tempi）进行综合理解

**偶然性维度**：

**必然性维度**：

**超自然维度**：

**时代维度**：
