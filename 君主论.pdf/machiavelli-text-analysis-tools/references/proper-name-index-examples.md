# 专名索引使用示例

## 人名查找示例

### 查找“切萨雷·博尔贾”

**索引条目**：`切萨雷·博尔贾：VII. 2, VII. 5, VII. 6, VII. 7, VII. 9, VII. 10, VII. 11, VII. 12, VII. 13, VII. 14, VII. 15, VII. 16, VII. 17, VII. 18, VII. 19, VII. 20, VII. 21, VII. 22, VII. 23, VII. 24, VII. 25, VII. 26, VII. 27`

**解读**：切萨雷·博尔贾在第7章的多个段落中出现（第2段至第27段），说明他是第7章的核心人物。

### 查找“教皇尤利乌斯二世”

**索引条目**：`教皇尤利乌斯二世：III. 3, VII. 5, VII. 6, VII. 7, XI. 1, XI. 2, XI. 3`

**解读**：教皇尤利乌斯二世在第3章、第7章和第11章中出现。
