# 《君主论》专名索引使用指南

## 使用场景
- 查找书中出现的人名（如切萨雷·博尔贾、教皇尤利乌斯二世）
- 查找地名（如佛罗伦萨、米兰）
- 查找民族名等专有名词
- 定位特定条目在书中的位置

## 索引结构

### 排序方式
- 索引条目按照**汉语拼音**排序

### 定位格式
- **罗马数字**：章节号
- **阿拉伯数字**：段落号

## 使用步骤

### 步骤1：确定查找目标
- 明确要查找的人名、地名或民族名
- 确认中文名称的准确拼写

### 步骤2：按拼音查找
- 根据汉语拼音在索引中定位条目
- 例如："阿尔巴"（a er ba）

### 步骤3：解读位置信息
- 格式："罗马数字. 阿拉伯数字"
- 含义：第X章第Y段
- 例如："VI. 3" 表示第6章第3段

### 步骤4：处理多重引用
- 若条目在多处出现，格式为："X. 1, X. 3"
- 表示分别出现在第X章第1段和第3段

## 示例

### 查找"切萨雷·博尔贾"
1. 拼音：qie sa lei · bo er jia
2. 在索引中定位
3. 假设结果："VII. 2, VII. 4, VIII. 1"
4. 解读：分别出现在第7章第2段、第7章第4段、第8章第1段

### 查找"佛罗伦萨"
1. 拼音：fo luo lun sa
2. 在索引中定位
3. 假设结果："III. 1, XXV. 2"
4. 解读：分别出现在第3章第1段、第25章第2段

## 注意事项
- 熟悉罗马数字与阿拉伯数字的对应关系
- 罗马数字范围通常为I-XXVI（第1-26章）
- 段落号根据具体版本可能有所不同
- 索引包含书中所有出现的人名、地名、民族名
