# Virtù 翻译详细说明

## 学术背景
国内学术界对「virtù」有多种译法：
- 德
- 德性
- 美德
- 德行
- 能力
- 才气
- 德行才干

本书采纳与申彤先生相近的译法，旨在涵盖马基雅维利笔下既包含道德力量又包含实际能力的复杂含义。

## 具体位置统计
- 全书近70处
- 包含3处「virtuoso」
- 包含1处「virtuosissimamente」

## 特殊语境详细说明

### 1. 古典含义语境
- **位置**：极少数与古典含义一致的地方
- **处理**：译为「德能」，同时附注〔德性〕
- **理由**：保留马基雅维利对古典传统的致敬

### 2. 技艺才能语境
- **位置**：第21章
- **处理**：译为「德能」，同时附注〔才能〕
- **理由**：此处强调实际技能而非道德品质

### 3. 善恶对立语境
- **位置**：第15章结尾
- **处理**：译为「善行」
- **理由**：与全书唯一一处出现的「恶行」形成直接对立

### 4. 诗歌语境
- **位置**：全书结尾处彼特拉克的诗句
- **处理**：直接译为「德性」
- **理由**：诗歌语言需要保持文学性和古典韵味

### 5. 事物语境
- **位置**：第6章开头、第14章开头
- **处理**：依据语境译出，并附注原文
- **理由**：用于非人的事物时，含义需根据具体语境调整
