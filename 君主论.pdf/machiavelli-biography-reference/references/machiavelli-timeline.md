# 马基雅维利生平大事年表

## 早期生涯与美第奇家族 (1469-1494)

- **1469年5月3日**：出生于佛罗伦萨
- **1498年6月19日**：被任命为佛罗伦萨共和国第二秘书厅秘书长
- **1498年7月14日**：被任命为"战争十人委员会"秘书

## 外交与政治活动 (1499-1512)

- **1500年7-11月**：第一次出使法国国王路易十二宫廷
- **1502年10月至次年1月**：第二次出使切萨雷·博尔贾
- **1506年12月6日**：建立国民军法案获得通过
- **1512年9月**：佛罗伦萨投降，美第奇家族返回，马基雅维利被免职

## 写作与晚年 (1513-1527)

- **1513年**：被捕入狱后获释，开始写作《君主论》
- **1515年左右**：开始创作《李维史论》
- **1520年11月8日**：接受委任撰写《佛罗伦萨史》
- **1525年5月**：将完成的《佛罗伦萨史》进献给教皇克莱门特七世
- **1527年6月21日**：去世，次日安葬于佛罗伦萨圣十字教堂
