# 继承与任命风险管理 - 详细案例

## 君主继承案例

### 卓越 → 软弱：成功继承

**努马之于罗穆卢斯**
- 前任：罗穆卢斯（卓越，罗马建立者）
- 继任者：努马（软弱，和平君主）
- 结果：努马享受罗穆卢斯的成果，成功维持王国

**所罗门之于大卫**
- 前任：大卫（卓越，统一以色列）
- 继任者：所罗门（软弱，依赖父辈遗产）
- 结果：所罗门享受大卫的成果，维持王国

**巴耶塞特之于穆罕默德**
- 前任：穆罕默德（卓越，奥斯曼扩张者）
- 继任者：巴耶塞特（软弱）
- 结果：巴耶塞特享受前任成果

### 软弱 → 软弱：王国毁灭

**罗波安无法维持所罗门的王国**
- 前任：所罗门（软弱）
- 继任者：罗波安（软弱）
- 结果：王国分裂，无法维持

**规律**：连续软弱君主必然导致王国毁灭

### 软弱 → 卓越：成就伟业

- 前任：软弱君主
- 继任者：卓越君主
- 结果：成就伟大事业，名声震天

## 人事任命案例

### 克劳狄乌斯·尼禄的危险任命

**背景**
- 克劳狄乌斯·尼禄曾受罗马城邦严重羞辱
- 被伤害后当选执政官

**危险策略**
- 意图：成功则恢复个人荣耀
- 后备：失败则报复城邦
- 风险：将个人复仇置于国家利益之上

**教训**
- 共和国绝不能将重要职务交给曾被严重伤害的人
- 受伤害者掌权必然带来报复风险

## 决策树逻辑

### 君主继承决策树
```
开始
  |
  ├─ 前任君主德能？
  |   ├─ 卓越 → 继任者德能？
  |   |   ├─ 卓越 → 维持并扩展王国
  |   |   └─ 软弱 → 尚可自保，享受成果
  |   |
  |   └─ 软弱 → 继任者德能？
  |       ├─ 卓越 → 成就伟大事业
  |       └─ 软弱 → 王国毁灭（高风险）
  |
  └─ 检查例外：法兰西王国？
      ├─ 是 → 依靠古老秩序维持
      └─ 否 → 应用上述规则
```

### 人事任命决策树
```
开始
  |
  ├─ 候选人曾被严重伤害？
  |   ├─ 否 → 可考虑任命
  ｜   └─ 是 → 禁止任命
  |
  └─ 若已任命 → 预警报复风险
      ├─ 成功 → 恢复个人荣耀
      └─ 失败 → 报复城邦
```

## 变量定义

| 变量名 | 类型 | 取值 | 说明 |
|--------|------|------|------|
| 前任君主德能 | Level | 卓越/软弱 | 前任统治者的能力水平 |
| 继任者德能 | Level | 卓越/软弱 | 继任者的能力水平 |
| 候选人受伤害史 | Boolean | 是/否 | 是否曾被国家严重伤害 |
