# 权力巩固战术

## 适用场景
- 新占领地区充满抢劫和动乱
- 必须实施残酷手段以自保
- 现任靠山（如教皇）即将离世且继任者可能敌对

## 战术一：利用代理人建立秩序与转移仇恨

### 执行步骤
1. **任命代理人**：选派一个残酷而干练的人，授予全权
2. **恢复秩序**：利用代理人的残酷手段在短时间内恢复和平与统一
3. **权力过渡**：在中心设立民政法庭，委派卓越庭长，取代代理人
4. **转移仇恨（关键步骤）：**
   - 抓住机会，在一个早晨将代理人处决
   - 公开展示尸体，放置象征暴力的道具
5. **达成效果：**
   - 人民感到心满意足（看到残酷者受惩）
   - 人民感到惶惑不安（畏惧君主的手段）
   - 涤荡怨气：表明过去的残酷行为来自大臣天性，而非君主本意

## 战术二：残酷手段的妥善使用

### 审视必要性
确定所有必须实施的侵害行为

### 执行原则（妥善使用）
- **时机：** 必须一次性完成（毕其功于一役）
- **目的：** 出于保护自己安全的必然性
- **后续：** 除非为了臣民的最大利益，否则绝不继续使用

### 避免模式（恶劣使用）
- **表现：** 虽然开始很少使用，但随着时间推移，不仅不停止反而日渐增加
- **后果：** 君主将无法维持自己的地位

### 心理效应管理
- **伤害：** 应一下子施加，被人品味得越少，造成的侵害越小
- **恩惠：** 应该点滴赐予，以便更好地被人品尝

## 战术三：应对政权更迭的四项防御措施

### 适用条件
- 现任支持者（如教皇）年老或即将去世
- 担心继任者对自己不友好

### 执行清单
1. **灭绝旧贵族血脉**：灭绝那些被废黜领主们的血脉世系
2. **笼络本地贵族**：把首都的所有贵族都笼络过来
3. **控制选举机构**：尽可能使枢机主教团倒向自己
4. **扩张实力**：在现任靠山未死之前获取更大的统治权

### 执行评估
在靠山去世时，检查这四件事的完成度。如果前三件已完成且第四件接近完成，则安全系数较高

## 关键变量

| 变量 | 类型 | 描述 |
|------|------|------|
| 代理人 | Role | 被授权执行残酷任务的代理人 |
| 公开展示 | Action | 处决代理人的公开仪式 |
| 残酷频率 | Pattern | 残酷手段的使用频率（一次性 vs 持续性） |
| 公共利益 | Boolean | 后续行为是否为臣民谋取利益 |
| 敌对家族 | Status | 敌对家族的生存状态 |
| 贵族忠诚度 | Status | 本地贵族的忠诚度 |
| 选举控制 | Percentage | 对决策机构的控制程度 |
| 领土实力 | Level | 自身拥有的军事和领土实力 |

## 预期结果

### 代理人策略
既恢复了秩序，又让人民对君主感恩戴德，同时消除了对君主的仇恨

### 残酷手段策略
将所有必要的残酷行动集中在极短时间内完成，随后转为施恩，以赢取民心

### 防御措施
通过消除内部威胁、控制政治盟友和增强自身实力，构建独立于特定靠山的防御体系
