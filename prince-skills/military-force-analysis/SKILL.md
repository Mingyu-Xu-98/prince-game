# 军队力量分析

## 适用场景
- 评估军队战斗力
- 与法兰西人作战或交往
- 组建或训练军队
- 制定对抗特定军队的战术

## 军队类型分类

### 类型一：激情四射且纪律严明（如罗马军队）
**特征**：
- 由纪律产生激情和德能
- 没有命令不得行动
- 不害怕危险
- 始终坚信胜利
- 愈战愈勇

**适用场景**：长期战争、持久战、需要高度组织的战役

### 类型二：有激情而无纪律（如法兰西军队）
**特征**：
- 依赖第一轮攻击的激情
- 开始时勇猛无比
- 若第一轮攻击未胜，激情消退
- 最终败退

**应对策略**：
- 抵挡首轮攻击
- 困住他们一段时间
- 稳操胜券

### 类型三：既无激情也无纪律（如意大利军队）
**特征**：
- 全无用处
- 除非敌人意外逃跑

**适用场景**：避免依赖此类军队

## 法兰西人特性分析

### 军事特性
**战斗模式**：
- 天性勇猛多于矫健
- 第一轮攻击暴风骤雨
- 若被有效抵挡，会松懈、疏忽职守
- 变得像女人一样胆怯

**战术应对**：
- 只要能困住他们一段时间，稳操胜券

**掠夺习惯**：
- 垂涎他人财货
- 抢劫后立即吃掉或用掉
- 甚至与被抢者共享
- （不同于西班牙人藏匿赃物）

### 政治/社会特性
**时间观**：
- 关注眼前利害
- 记不得过去恩惠
- 不在意未来

**协议观**：
- 最初达成的协议总是最好

**行为模式**：
- 许诺容易，履行勉强
- 运道不好时谦卑，好时傲慢
- 喜爱金钱甚于流血
- 慷慨只在大庭广众之下

### 宫廷生存法则
- 需腰缠万贯
- 孜孜不倦
- 好运
- 对评论置若罔闻
- 不太在意荣誉

## 军队组建建议
1. 追求罗马式纪律型军队
2. 避免仅依赖激情的军队
3. 建立严格的指挥体系
4. 培养士兵的长期战斗意志
