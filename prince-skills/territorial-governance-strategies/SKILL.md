# 领土治理策略

## 适用场景
- 计划征服或已获得新领土
- 需要巩固对异质文化地区的统治
- 面临世袭君主国的维持问题
- 需要评估不同政治结构国家的征服与保有难度

## 世袭君主国维持策略

### 核心原则
- 维持统治比在新国家困难小得多，只需付出寻常努力
- 不背离先人定下的秩序（保持传统制度）
- 遇到意外事件时能够随机应变

### 稳定性分析
- 自然/世袭的君主侵害人民的理由较少，更受爱戴
- 除非有超乎寻常的恶行使君主为人憎恨，否则民众自然会有好感
- 革新的记忆与理由因统治的古老和持续而消除

### 风险阈值
- 只有遇到超乎寻常、格外强大的势力，才可能被篡夺
- 一旦篡夺者发生祸患，君主还能收复失地

## 混合君主国稳固策略

### 同质混合君主国（同语言同风俗）

**适用条件：**
- 被征服国家与征服国属于同一地区
- 使用同一语言
- 被征服者不习惯于自由的生活

**执行步骤：**
1. 灭绝过去统治它们的君主的血脉世系
2. 在其他事情上维持它们旧的状态
3. 不改变它们的法律或赋税

**预期结果：** 领地在非常短的时间内与已有君主国浑然一体

### 异质混合君主国（异语言异风俗）

**策略一：君主亲自驻跸**
- 动乱稍一露头就能察觉并迅速补救
- 防止官吏掠夺
- 臣民可随时求助而感到满意
- 良民更爱戴，坏人更畏惧
- 外部进攻者更加迟疑

**策略二：派遣殖民（推荐）vs 驻军（避免）**
- **殖民优势：** 花费不高、更加忠诚可靠、侵害性小
- **驻军劣势：** 靡费甚巨、军队危害整个国家

**策略三：地缘政治制衡**
- 成为邻近弱小势力的首领和保护者
- 削弱该地区的强大势力
- 防止同自己一样强大的外国势力介入

**策略四：预防性干预**
- 在患病之初（忧患未萌）就察识并恢复
- 绝不为了避免一场战争而听任动乱继续
- 宁愿在自己有能力时在远方作战，也不愿将来不得不在家门口作战

**禁忌法则：** 谁是促使他人强大的原因，谁就是自取灭亡

## 不同类型国家的征服与保有策略

### 类型A：由一位君主和臣仆统治（如土耳其苏丹）
- **征服难度：** 困难（无法通过收买内部人员进入）
- **保有难度：** 容易（彻底击败君主并灭绝其血脉世系即可）

### 类型B：由一位君主和诸侯统治（如法兰西国王）
- **征服难度：** 相对容易（可收买心怀不满的诸侯）
- **保有难度：** 非常困难（残留领主仍会成为新变革的首领）

### 习惯自由的城市/国家（共和国）
**策略选择（三种方式）：**
1. **毁灭它们**（最安全稳固）
2. **亲自前往生活在那里**
3. **允许它们生活在自己的法律下，但建立寡头政权**（要求进贡并建立依赖君主的寡头政府）

**警告：** 如果不毁灭它们，它们最终会摧毁统治者

### 习惯于君主统治且君主血脉已灭绝
- **优势：** 习惯服从，不知道如何自由生活
- **策略：** 轻而易举地获得并保有

## 决策流程

1. **识别领土类型：** 世袭君主国 / 同质混合君主国 / 异质混合君主国 / 自由城市 / 习惯君主统治的国家
2. **评估政治结构：** 君主-臣仆型 / 君主-诸侯型
3. **选择对应策略：** 参考上述各类型的具体执行方案
4. **执行并监控：** 根据反馈调整策略
