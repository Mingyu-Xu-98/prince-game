# 安全与军事策略

## 使用场景
- 存在潜在的阴谋者
- 拥有一定民众基础
- 需要同时应对军队和民众的需求
- 拥有国家或领土
- 面临内部或外部安全威胁
- 需要稳固国家
- 获取新领土

## 第一部分：防范阴谋的民众支持策略

### 1. 核心防御机制
避免受到人民普遍的憎恨和鄙视，让人民感到满意。

### 2. 阴谋者的心理障碍

**心理分析：**
- 阴谋者通常认为杀死君主能满足人民
- 如果阴谋者相信弑君只会得罪人民，他们就没有勇气采取行动

### 3. 阴谋实施的困难（不对称性）

**阴谋者方面：**
- 除了恐惧、猜疑、担心令人丧胆的惩罚之外，什么都没有

**君主方面：**
- 拥有君权的威严
- 拥有法律
- 拥有盟友
- 拥有国家的防卫

### 4. 揭露风险

**风险机制：**
- 阴谋不可能单枪匹马，必须寻找同伙（通常是心怀不满者）
- 一旦向同伙透露意图，就给了对方通过告密获得收益的机会

### 5. 结果导向的民众心理
人们只看结果。只要君主赢得并维持国家，手段就被认为光荣。

## 第二部分：军队与民众的平衡管理

### 1. 识别冲突需求

**人民需求：** 热爱和平安宁，喜爱温和谦逊的君主

**士兵需求：** 喜爱尚武、傲慢、残酷、贪婪的君主，以便获得加倍军饷和满足贪欲

### 2. 罗马皇帝的特殊困境（三方博弈）

**困境：** 除了大人物和人民，还必须容忍士兵的残酷与贪婪

**新皇帝的困境：** 往往被迫优先满足士兵，导致损害人民，从而招致仇恨

### 3. 成功案例分析

**马尔库斯（世袭）：**
- 德能高
- 让士兵和人民各安其位
- 未招憎恨或鄙视

**塞维鲁（新君主）：**
- 极其残酷但德能高
- 利用士兵威慑人民
- 虽受人民痛恨但能统治

### 4. 失败案例分析

**佩蒂纳克斯/亚历山大：**
- 试图行善但软弱
- 被士兵蔑视和杀害

**康茂德/马克西米努斯：**
- 既被人民憎恨又被士兵/人民蔑视
- 被杀

### 5. 现代君主的策略调整

**一般原则：** 现代君主（除土耳其和苏丹外），人民比士兵更有力量，因此满足人民比满足士兵更有必要。

**例外情况：**
- **土耳其皇帝**：拥有12000名步兵和15000名骑兵的常备军，王国安全完全依赖他们，必须优先与军队保持友好
- **苏丹**：国家掌握在士兵手中，不管人民如何，必须与军队保持友好

### 6. 新君主的行动指南

- 不能完全仿效马尔库斯（世袭且德能高）
- 也不必完全照搬塞维鲁（极其残酷）
- 应从塞维鲁那里吸取奠定国家基础所需的成分
- 从马尔库斯那里吸取维护稳固国家并获得荣耀的成分

## 第三部分：军事控制与堡垒建设策略

### 1. 武装臣民策略（针对新君主）

**核心原则：** 绝不要解除臣民的武装；相反，如果发现他们没有武装，就将其武装起来。

**理由：**
- 武装后的军队会变成你的军队
- 原本怀疑的人变得忠诚，原本忠诚的人保持忠诚
- 臣民转变为拥戴者

**操作细节：**
- 由于无法武装所有人，武装一部分人会使他们感到受恩惠
- 受优待者因承担更大危险而理应获得更多，其他人也会谅解

**解除武装的后果：**
- 被视为侵害
- 表明君主不信赖臣民（认为其怯懦或无信义）
- 会引发仇恨
- 解除武装后不得不依赖不可靠的雇佣军

### 2. 解除新领土武装策略

**原则：** 当君主获取一个新国家（如同新肢体嫁接到旧肢体上）时，必须解除该国的武装。

**例外：** 除了在获取该国时拥戴你的人。

**对例外者的处理：** 必须抓紧时间让他们变得柔顺和懦弱。

**最终目标：** 确保所有武器只掌握在你自己的士兵手中，这些士兵应生活在你的旧国并拱卫在你近旁。

### 3. 派系纷争策略（分而治之）

**历史做法：** 古人（如保有皮斯托亚和比萨）主张在城镇制造内部分歧以便更容易保有。

**现代建议：** 不要奉此为箴规。

**理由：**
- 一旦敌人迫近，分裂的城市必然立即丧失
- 较弱小的派系总是投靠外国势力
- 另一派系则无法统治
- 这种方法显示君主的虚弱，只有在和平时期有利可图，战争到来时只会显示其谬误

### 4. 堡垒建设策略

**用途：** 作为对付反对者的笼头和应对突然袭击的避难所。

**决策逻辑：**
- IF (害怕人民甚于害怕外国人) THEN 应当兴建堡垒
- IF (害怕外国人甚于害怕人民) THEN 应当放弃堡垒

**历史案例：**
- **尼科洛·维泰利和乌尔比诺公爵**：摧毁堡垒，认为没有堡垒再度失去国家更难
- **弗朗切斯科·斯福尔扎**：城堡带来的战争多于其他动乱

**核心原则：** 最好的堡垒是不被人民憎恨。

- 如果拥有堡垒但被人民憎恨，任何堡垒都救不了你（人民拿起武器，外国人会帮助）

**例外情况：** 弗利伯爵夫人凭借堡垒躲过民众暴动并等待米兰援助（当时外国人无法帮助人民）。但当切萨雷·博尔贾进攻且人民与外国人联合时，堡垒便失去了价值。

## 关键变量

- **民众态度**：满意/仇恨/鄙视
- **阴谋者风险**：背叛风险/惩罚风险
- **时代背景**：古罗马时代 / 现代君主国 / 土耳其苏丹国
- **优先支持对象**：士兵 / 人民
- **臣民忠诚度**：臣民对君主的信任和支持程度
- **威胁来源**：主要威胁来自内部人民还是外部势力
- **领土状态**：旧领土或新获取的领土

## 输出结果

若获得民众好感，阴谋难以成功；若民众抱有敌意，则需恐惧任何人；根据政体特征，优先满足掌握实际力量的一方（现代为人民，特殊政体为军队）；决定是否武装/解除武装、是否制造派系对立、是否修建或摧毁堡垒的具体行动指南。
