# 决策框架详解

## 权力基础选择矩阵

| 维度 | 依靠人民 | 依靠大人物 |
|------|----------|------------|
| 地位稳固性 | 高 | 低 |
| 满足难度 | 低 | 高 |
| 忠诚度 | 高 | 低 |
| 背叛风险 | 低 | 高 |
| 逆境表现 | 可靠 | 危险 |

## 大人物分类决策树

```
大人物
├─ 完全仰赖君主机运
│  └─ 策略：赐予荣誉和宠爱
│     └─ 警惕：贪得无厌者
└─ 不仰赖君主机运
   ├─ 出于胆怯
   │  └─ 策略：利用，特别是建议者
   └─ 出于野心
      └─ 策略：当作公开敌人防范
```

## 政权更迭风险评估

### 四项措施的权重
1. 灭绝旧贵族血脉：消除内部威胁（高权重）
2. 笼络本地贵族：建立政治盟友（高权重）
3. 控制选举机构：确保政治支持（高权重）
4. 扩张自身实力：增强独立能力（中权重）

### 安全系数计算
- 前三项完成 + 第四项接近完成 = 高安全系数
- 前两项完成 + 后两项部分完成 = 中安全系数
- 仅第一项完成 = 低安全系数
- 全部未完成 = 极高风险
