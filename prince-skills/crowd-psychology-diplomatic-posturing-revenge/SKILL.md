# 群众心理、外交姿态与复仇机制

## 适用场景
- 面对群众反抗或骚乱
- 应对傲慢无理或心怀仇恨的敌人
- 发生伤害事件需要决定报复策略
- 共和国或君主需要预防内部危机

## 群众心理分析与控制

### 群众状态评估
1. **联合状态**：群众联合在一起时强大凶悍
2. **分散状态**：分散开来时弱小驯服
3. **无首领状态**：既可怕又虚弱，缺乏组织
4. **有首领状态**：团结一致，会思考防卫

### 控制策略
- 利用群众相互不信任的特点，通过展示眼前惩罚迫使其争先恐后服从
- 当群众失去自由或失去爱戴的君主时，会产生极度危险的坏性情，需要强力补救
- 分化群众，防止选出统一首领

## 外交姿态策略

### 应对傲慢敌人
- **禁止谦卑**：对傲慢无理或心怀仇恨的人使用谦卑会带来损害，增加其狂妄
- **绝不放弃**：除非能保有，否则绝不应自愿放弃某物
- **战争选择**：让人凭武力夺走比因害怕而任人夺走更好
- **分化策略**：当有多个敌人时，给予某一人东西以争取过来，断绝其与其他敌人的联盟

### 战争与和平原则
- 避免战争往往导致更多战争
- 暴露怯懦会招致更多侵略
- 必须展示武力而非软弱

## 复仇机制

### 核心规则
共和国或君主不对集体或个人受到的伤害进行报复是危险的。

### 复仇必要性
- 若受伤害者未获补偿，会设法报复哪怕导致共和国毁灭
- 若在君主统治下，受伤害者会不惜代价报复君主

### 执行原则
- 必须及时报复伤害以避免毁灭性反击
- 不得忽视伤害或反而奖励施暴者
- 案例警示：罗马人未惩罚法比乌斯三兄弟反而授以荣誉，导致法兰西人进攻并占领罗马

## 决策流程
1. 评估群众状态（联合/分散/有首领/无首领）
2. 判断敌人态度（傲慢/仇恨）
3. 确定伤害处理方式（报复/忽视/奖励）
4. 应用相应策略
