# 马基雅维利参考文献使用指南

## 专名索引使用规则

### 排序方式
- 索引条目按照汉语拼音排序

### 定位格式
- **罗马数字**：章节号
- **阿拉伯数字**：段落号

### 示例解读
- 「阿尔巴：VI. 3」 → 第6章第3段
- 「X. 1, X. 3」 → 第X章第1段和第3段

### 内容覆盖
- 人名（如切萨雷·博尔贾、教皇尤利乌斯二世）
- 地名（如佛罗伦萨、米兰）
- 民族名等

### 意大利文版
适用于学术研究和版本考证，共6种主要版本。

### 英译本
适用于英语读者和学术研究，共10种主要译本。

### 中译本
适用于中文读者和翻译对比研究，共10种主要译本。

### 研究译著
适用于马基雅维利研究和政治思想史研究。

## 执行步骤

1. **确定需求**：明确需要查阅的文献类型
2. **选择版本**：根据研究目的选择合适的版本
3. **使用索引**：按拼音查找条目，按罗马数字定位章节
4. **对比版本**：必要时对比不同版本的翻译差异
5. **查阅研究**：参考研究译著深化理解

## 注意事项

- 不同版本有不同编者和特点
- 不同译本有不同侧重点
- 不同中译本依据不同英译本底本
- 需熟悉罗马数字与阿拉伯数字的对应关系
