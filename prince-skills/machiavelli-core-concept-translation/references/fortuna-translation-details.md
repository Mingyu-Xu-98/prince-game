# Fortuna 翻译详细说明

## 译法选择理由

### 旧译法
- 过去通常译为「命运」

### 新译法
- 采纳徐卫翔教授建议
- 译为「机运」

### 理由
- 突出其偶然性、流转性的面相
- 区别于表达必然性的「fatum」

## 频率统计
- 全书出现50余次
- 约17次与「德能」连用

## 特定短语翻译详情

### privata fortuna
- **出现次数**：全书5处
- **翻译**：「平民运道」
- **理由**：强调非官方、非贵族的运道属性

### astuzia fortunate
- **出现次数**：全书1处
- **翻译**：「幸运的机巧」
- **理由**：结合「astuzia」（机巧）与「fortuna」（机运）的双重含义
