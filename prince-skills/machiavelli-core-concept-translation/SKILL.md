# 马基雅维利核心概念翻译

## Virtù（德能）翻译规则

### 基本规则
- **用于人时**：一律译为「德能」
- **覆盖范围**：全书近70处，包含3处「virtuoso」和1处「virtuosissimamente」

### 特殊语境处理
| 语境类型 | 翻译方式 | 备注 |
|---------|---------|------|
| 古典含义 | 「德能」〔德性〕 | 极少数与古典含义一致处 |
| 技艺才能 | 「德能」〔才能〕 | 第21章一处 |
| 善恶对立 | 「善行」 | 第15章结尾，与「恶行」相对 |
| 诗歌语境 | 「德性」 | 全书结尾彼特拉克诗句 |
| 用于事物 | 依语境译出+附注原文 | 第6章、14章开头 |

## Fortuna（机运）翻译规则

### 基本规则
- **标准译法**：一律译为「机运」
- **拟人化**：译为「机运女神」
- **出现频率**：全书50余次，约17次与「德能」连用

### 特定短语翻译
- `privata fortuna` → 「平民运道」
- `astuzia fortunate` → 「幸运的机巧」

## 命运相关术语语义层级

### 术语分类与必然性强度（递减）
1. **fatum/fato**：表达必然性的命运
2. **destino**：表达必然性的命运
3. **sorte**：用法灵活，译为「运气」，标注原文
4. **fortuna**：表达盲目偶然的外在力量

### 特殊情况
- 表达必然性时，有时也用「sorte」
- 区别于必然性的「fatum」

## 书信121外在力量概念群

当分析书信121「佩鲁贾的奇思妙想」时，识别以下六个核心概念：
- fortuna（机运）
- sorte（运气）
- fati（命运）
- tempi（时势）
- cielo（上天）
- stele（星宿）
